# Lucky Station v3.0 更新公告

本次更新重点围绕 **新增玩法、福利体验、游戏结算稳定性和后台运营能力** 展开。玩家侧体验更顺畅，管理员侧也能更清楚地查看数据、处理反馈和管理活动。

## ✨ 新增内容

### 1. 新增 2048 小游戏

游戏中心新增 **2048**，目前游戏中心共开放 **7 款挑战**。

- 采用 5x5 棋盘
- 支持滑动合成、冲击 2048 目标
- 支持得分、最高方块、有效步数和预估积分展示
- 单局积分根据得分和里程碑计算，最高可获得 500 福利积分
- 刷新或中断后可恢复当前游戏进度

### 2. 多人抽奖支持“到点开奖”

多人抽奖新增按时间自动开奖模式。

- 活动页会展示开奖时间
- 到点后系统自动开奖，无需管理员手动守着页面
- 首页、福利商店和抽奖详情页都会显示对应开奖状态

### 3. 公告支持 Markdown 排版

站内公告现在支持 Markdown 内容展示。

- 支持标题、加粗、列表、引用、代码块、链接等格式
- 首页公告弹窗展示更清晰
- 公告也会同步进入通知中心，方便用户回看

## 🎮 游戏体验优化

- 优化多款小游戏的结算流程，网络波动或重复点击时更不容易漏发或重复发放积分
- 强化游戏服务端权威结算，异常情况下可重试或同步最新状态，减少漏发和重复发放
- 优化 2048、连连看、消消乐、记忆翻牌、扫雷、打地鼠、星尘迷阵等游戏的服务端校验与记录逻辑
- 改善移动端和内嵌浏览器兼容性，页面高度、触控、动画和资源加载更稳定

## 🎁 福利与账户优化

- 福利项目更偏向直接发放站内积分，领取后积分到账更直观
- 提现和充值流程增强自动核对能力
- 如果账户额度或积分状态暂时不确定，系统会自动尝试确认、补发、退款或回滚
- 开心农场购买种子、道具等场景增加异常补偿，减少“扣了积分但没到账”的情况

## 🌱 环保行动优化

- 环保奖品发放与偷取流程更稳定
- 优化奖品归属核对和异常恢复，降低奖品归属错乱风险
- 修复跨用户操作异常时的状态回滚问题，降低奖品归属错乱风险
- 增加手动回收统计，为后续活动和榜单提供更准确的数据基础

## 🔔 通知与反馈优化

- 反馈状态变化、管理员回复、钱包到账等通知更完整
- 管理员可更高效地处理反馈内容
- 异常或无效反馈支持后台删除，反馈列表更干净

## 🛠 后台管理升级

### 运营看板

- 新增今日活跃、签到、游戏参与、反馈待办等运营指标
- 新增积分收入与支出分析
- 支持按日、按周、按月查看积分流向
- 增加风险告警入口，方便发现异常积分增长、异常抽奖频次等情况

### 用户管理

- 用户列表新增积分余额、今日游戏、今日积分、最近活跃等信息
- 用户详情页可查看积分流水、福利领取、抽奖记录、最近游戏、通知和成就
- 支持管理员手动调整积分
- 支持管理用户“奉献者”等成就状态

### 环保管理

- 新增环保管理后台
- 可查看奖品概率、奖品持有人、偷取记录和手捡统计
- 支持调整环保奖品出现概率

## ✅ 稳定性修复

- 修复环保行动跨用户状态回滚问题
- 修复农场购买异常时的积分补偿问题
- 优化钱包提现/充值结果不确定时的自动恢复
- 优化小游戏结算重复提交、断线恢复和异常状态同步
- 增加定时维护任务，自动处理到期项目和到点抽奖

## 📌 使用提示

- 2048 可从「游戏中心」进入
- 到点开奖活动不需要停留在页面，开奖后刷新即可查看结果
- 如果提现、充值或农场购买出现短暂处理中状态，可稍后刷新页面，系统会自动核对并处理
